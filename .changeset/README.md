# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`:

```sh
pnpm changeset
```

To record a change that affects a published package, add a Markdown file inside `.changeset/` describing the change. The Release GitHub Action will then open a version PR (or publish directly on `main` once the version PR is merged).

See [`CONTRIBUTING.md`](../CONTRIBUTING.md#release-process) for the full release workflow.
