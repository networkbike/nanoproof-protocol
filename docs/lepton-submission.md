# Lepton Hackathon Submission Checklist

> Use this checklist before submitting to the [Lepton Agents Hackathon](https://lepton.thecanteenapp.com/) (deadline: **Jul 6, 2026, 11:59 PM ET**).

---

## Pre-submission

- [ ] Public GitHub repo is clean, well-documented, and shows clear progress since Jun 15
- [ ] Repository has top-level `README.md`, `ARCHITECTURE.md`, `ROADMAP.md`, `CONTRIBUTING.md`, `SECURITY.md`
- [ ] All public packages have their own README
- [ ] `.env.example` files present in every app + package
- [ ] Code commits after Jun 15 are visible (commits, not squashed merges)
- [ ] Live deployed link works without login (judges should be able to demo)

## Submission form (forms.gle/SMqLaw2pMGDe58LFA)

- [ ] GitHub repo URL filled in correctly
- [ ] Demo video URL filled in (≤3 minutes)
- [ ] Live deployed link filled in (strongly encouraged)
- [ ] Traction question: how many users onboarded — answered with concrete numbers
- [ ] Traction question: what user problem you're solving — answered with concrete story

## Demo video script (≤3 min)

| Time | Beat |
|------|------|
| 0:00 | What is NanoProof? (1 sentence) |
| 0:15 | The problem: creators get nothing when AI cites them |
| 0:30 | The fix: instant USDC nanopayments on Arc |
| 0:45 | Live demo: an AI agent answers a question with citations |
| 1:30 | Live demo: watch testnet USDC flow to a creator wallet |
| 2:15 | Why this matters beyond the hackathon |
| 2:45 | Call to action + what's next |

## Traction story

Make it concrete:

- ❌ "We're building a great product"
- ✅ "We onboarded 12 creators during the Lepton window. Across 80 AI-generated responses, the protocol settled 1,400 citations totaling $1.27 in testnet USDC. The top creator earned $0.34."

## Returning builders

If this is your second Canteen event, the judges will **only score progress after Jun 15**. Make sure:

- [ ] New commits after Jun 15 are visible
- [ ] The repo's commit history isn't a single giant commit
- [ ] The README has an "Update since Agora" or "Progress since Jun 15" section
- [ ] You submitted via the Luma with code `AGORA_RETURNx1313` (NOT the website's `LEPTONHOUSE` code)

## Submission day checklist

- [ ] Submit early — you can resubmit unlimited times before the deadline
- [ ] Re-verify all URLs in the form (typos are the #1 reason for disqualification)
- [ ] Re-test the live deployed link from an incognito window
- [ ] Watch your own demo video start-to-finish — no audio glitches, no broken frames
- [ ] Introduce yourself in `#lepton-hackers` Discord with your repo + deployed link
